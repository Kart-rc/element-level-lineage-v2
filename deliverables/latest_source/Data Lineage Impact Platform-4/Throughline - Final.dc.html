How lineage is collected
No single source is complete. A baseline is extracted from code by static analysis + an LLM, kept current by a CI/CD diff on every change, and corroborated at runtime by Spark/Dask collectors and OTel traces — every edge carries a confidence score from the signals that agree.

How it scales
1Hierarchy — Org → Domain → Service → Dataset → Column. Descend one altitude at a time.
